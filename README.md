Игра «Инопланетное вторжение»
Мой первый полноценный проект на Python, созданный по книге «Изучаем Python» Эрика Мэтиза. Это классическая 2D-аркада, в которой нужно защищать Землю от флота пришельцев.
Управление: Стрелки для движения, Пробел для стрельбы.
Особенности: В игре реализована система уровней, подсчет очков и постепенное повышение сложности.
Технологии: Python, библиотека Pygame.
